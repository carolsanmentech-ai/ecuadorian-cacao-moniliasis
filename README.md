# ecuadorian-cacao-moniliasis
Trabajo Fin de Máster (UNIR): Detección de moniliasis (*Moniliophthora roreri*) en cultivos de cacao ecuatoriano mediante técnicas de visión artificial. Incluye pipeline en cascada (YOLO11 + ResNet50+CBAM) y segmentación con SAM2 + ConvNeXt, con explicabilidad mediante Grad-CAM e IoU
